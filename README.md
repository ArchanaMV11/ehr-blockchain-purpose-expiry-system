# ehr-blockchain-purpose-expiry-system
Purpose-Bound and Expiry-Aware Blockchain System for EHR Management
Problem Statement

Traditional EHR systems face several issues:

Unauthorized or excessive data access
Lack of fine-grained access control
No automatic expiration of permissions
Risk of data tampering

Proposed Solution

This system uses blockchain principles to:

Provide secure and tamper-proof record storage
Enforce purpose-specific access restrictions
Implement time-bound (expiry-based) permissions
Maintain transparency and traceability of data access

Key Features
Secure EHR data handling
Expiry-based access control
Purpose-specific authorization
Immutable transaction logging (blockchain concept)
Role-based access (Doctor / Patient / Admin)
Audit trail for tracking data usage
